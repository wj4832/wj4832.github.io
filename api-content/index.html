If chars is given and not None, remove characters in chars instead. """ pass 方法注释的意思是：当你使用strip方法时，如果不传入参数，那么此时会把你想要处理的字符串对象，去除首尾的空格后返回结果； 如果传入了参数，且参数不是None的时候，那么就会在被处理字符串的首尾分别删去你传入的参数字符串中每一个元素，然后再返回。 所以，上面的代码中，a_string.strip("ap")并不是删去首尾的"ap"字符，而是在首尾处分别查找是否存在字符"a"或者"p"，一直删除到字符串不以"a"或者"p"开头/结尾的时候才停止，然后返回结果。 解决方案 使用removeprefix删除字符串开头特定字符，removesuffix删除结尾特定字符（推荐）。如： a_string = "append this list with string paap" resp = a_string.removeprefix("ap").removesuffix("ap") print(resp) # 预期结果 pend this list with string pa # 实际结果 pend this list with string pa 使用re.sub方法，在正则表达式中指定开头和结尾的字符串。如： a_string = "append this list with string paap" resp = resp = re.sub("^ap|ap$", "", a_string) print(resp) # 预期结果 pend this list with string pa # 实际结果 pend this list with string pa 以上两种方法更推荐第一种方法，因为无论是在简易程度还是运行速度方面，第一种方法都要明显更好于第二种方法。 测试代码如下： import re import time a_string = "append this list with string paap" loop_count = 1000000 def a(): start_time = time.time() for i in range(loop_count): re.sub("^ap|ap$", "", a_string) print("正则方法用时：", time.time()-start_time) def b(): start_time = time.time() for i in range(loop_count): a_string.removeprefix("ap").removesuffix("ap") print("remove方法用时：", time.time()-start_time) a() b() # 正则方法用时： 1.4415099620819092 # remove方法用时： 0.09711885452270508 ","link":"https://wj4832.github.io/post/ni-ke-neng-dui-python-de-strip-fang-fa-you-yi-xie-wu-jie/"}]}
